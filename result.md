## 2. Обзор организаций

---

### 2.1. ISOC (Internet Society) — Общество Интернета

| | |
|---|---|
| **Тип организации** | Международная профессиональная организация |
| **Год основания** | 1992 год |
| **Основная деятельность** | Обеспечение открытого развития и доступности интернета. Предоставляет организационную основу для IETF и IAB |
| **Официальный сайт** | [https://www.internetsociety.org/](https://www.internetsociety.org/) |

**Краткое описание:**  
ISOC объединяет более 20 000 индивидуальных членов и более 100 организаций из 180 стран мира. Организация занимается вопросами доступности интернета, безопасности и развития сетевых технологий.

---

### 2.2. IRTF (Internet Research Task Force) — Исследовательская группа интернет-технологий

| | |
|---|---|
| **Тип организации** | Исследовательское подразделение IAB |
| **Год основания** | 1989 год |
| **Основная деятельность** | Долгосрочные исследования в области архитектуры интернета, разработка новых протоколов и сетевых приложений |
| **Официальный сайт** | [https://irtf.org/](https://irtf.org/) |

**Краткое описание:**  
IRTF фокусируется на фундаментальных исследованиях, результаты которых в будущем могут быть использованы IETF для создания новых стандартов.

---

### 2.3. IETF (Internet Engineering Task Force) — Инженерный совет Интернета

| | |
|---|---|
| **Тип организации** | Открытое международное сообщество инженеров и разработчиков |
| **Год основания** | 1986 год |
| **Основная деятельность** | Разработка стандартов протоколов интернета (RFC), архитектуры TCP/IP |
| **Официальный сайт** | [https://www.ietf.org/](https://www.ietf.org/) |

**Краткое описание:**  
IETF является основной организацией, отвечающей за технические стандарты интернета. Все документы проходят открытое обсуждение в рабочих группах.

---

### 2.4. IEEE (Institute of Electrical and Electronics Engineers) — Институт инженеров электротехники и электроники

| | |
|---|---|
| **Тип организации** | Некоммерческая инженерная ассоциация (США) |
| **Год основания** | 1963 год |
| **Основная деятельность** | Разработка стандартов в области электротехники, электроники и компьютерных сетей (Ethernet 802.3, Wi-Fi 802.11) |
| **Официальный сайт** | [https://www.ieee.org/](https://www.ieee.org/) |

**Краткое описание:**  
IEEE разрабатывает стандарты, используемые в аппаратном обеспечении компьютеров и сетевом оборудовании по всему миру.

---

### 2.5. ICANN (Internet Corporation for Assigned Names and Numbers) — Корпорация по управлению доменными именами и IP-адресами

| | |
|---|---|
| **Тип организации** | Международная некоммерческая организация |
| **Год основания** | 1998 год |
| **Основная деятельность** | Распределение IP-адресов, управление корневыми DNS-серверами, регулирование доменных имен |
| **Официальный сайт** | [https://www.icann.org/ru](https://www.icann.org/ru) |

**Краткое описание:**  
ICANN обеспечивает уникальность идентификаторов в интернете. С 2016 года является полностью независимой международной организацией.

---

### 2.6. EIA (Electronic Industries Alliance) — Альянс электронной промышленности

| | |
|---|---|
| **Тип организации** | Профессиональная организация (США) |
| **Год основания** | 1924 год |
| **Основная деятельность** | Разработка стандартов для электронных компонентов и интерфейсов (RS-232, RS-485) |
| **Официальный сайт** | [https://en.wikipedia.org/wiki/Electronic_Industries_Alliance](https://en.wikipedia.org/wiki/Electronic_Industries_Alliance) |

**Краткое описание:**  
EIA разрабатывает стандарты (RS — Recommended Standards), используемые в электронной промышленности и компьютерной технике.

---

### 2.7. ITU (International Telecommunication Union) — Международный союз электросвязи

| | |
|---|---|
| **Тип организации** | Специализированное учреждение ООН |
| **Год основания** | 1865 год (как Международный телеграфный союз) |
| **Основная деятельность** | Регулирование использования радиочастот, стандартизация телекоммуникаций |
| **Официальный сайт** | [https://www.itu.int/ru/](https://www.itu.int/ru/) |

**Краткое описание:**  
ITU является старейшей международной организацией в сфере связи, определяющей глобальные стандарты для телекоммуникационных технологий.

## 4. Вывод

В ходе выполнения лабораторной работы были изучены основные организации, занимающиеся стандартизацией интернета. Каждая организация выполняет свою уникальную роль:

* **ISOC** — координирует развитие интернета на глобальном уровне
* **IETF и IRTF** — разрабатывают технические стандарты и проводят исследования
* **IEEE** — создает стандарты для сетевого оборудования
* **ICANN** — управляет адресным пространством интернета
* **EIA и ITU** — регулируют вопросы электроники и международной связи

Благодаря деятельности этих организаций обеспечивается совместимость устройств и протоколов по всему миру, что позволяет интернету функционировать как единая глобальная сеть.
